Modo IA generativa

Si RETAsist está publicado en un servidor (intranet o web), la IA viene activada automáticamente contra /api/asistente de esa misma web: los agentes no configuran nada. Si la abres como archivo, funciona sin conexión, y aquí puedes indicar la URL de un proxy propio (la clave de API vive en el servidor, nunca en este archivo).

Ejemplo de proxy

Reenvía el cuerpo a la API de Anthropic añadiendo tu clave del lado servidor. Soporta streaming: si reenvía el flujo SSE tal cual, la respuesta aparece token a token.

El proxy debe permitir CORS desde el origen donde abras RETAsist (incluido el preflight OPTIONS). El asistente envía stream: true: si el proxy reenvía el flujo, verás la respuesta token a token; si devuelve JSON normal, también funciona. Si el proxy falla, el asistente vuelve solo al modo offline.
